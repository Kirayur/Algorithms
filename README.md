# Решение задачи в рамках курса по алгоритмам и структурам данных на ПИ ВШЭ

В рамках последнего пункта задачи воспользуемся статьёй https://algo.inria.fr/flajolet/Publications/FlFuGaMe07.pdf

Улучшение заключается в использовании корректирующей функции для сырых оценок, чтобы приближать их к истинным значениям более точно.
